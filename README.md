# University Management System

## Description
A Python GUI application built with Tkinter and SQLi to help universities manage student accounts, records, and communication efficiently. This project demonstrates skills in Python programming, database management, and GUI development.

## Features
- **Student Registration & Login**: Students can create accounts and securely log in.
- **Student ID Card Generation**: Automatically generates digital student ID cards.
- **Password Management**: Students can change passwords securely.
- **Record Management**: Admin can edit, delete, and manage student records.
- **Admin Student Search**: Search students by name, class, or student ID.
- **Class Announcements**: Admin can send announcements to specific classes.

## Built With
- Python 3
- Tkinter (GUI development)
- SQLite (Database management)
- Pillow (PIL) (Image processing)

## How to Run
1. Install Python 3.x
2. Install required libraries: `pip install -r requirements.txt`
3. Run `home.py` to start the application
4. Make SQLite database is set up as per configuration
